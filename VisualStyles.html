How Graphics have advanced

Photorealistic games try to be as realistic as possible from the way the world looks to the way wind interacts with a person’s hair. Some photorealistic games now go as far as recreating entire city’s in the game down to the smallest details a prime example of this is Ubisoft’s latest game The Division 2 which is set in a perfectly recreated Washington DC.

Cel-shading is a style used to make a 3D object seem as if it is almost 2D. It does this by replacing the shaded colors with flat colors. This visual style has even inspired people when cosplaying to recreate the looks of Cel-shading with makeup and paints.

The most resent abstract/unique visual style used in a video game would be the style used in the game “Curse of the Obra Din”. This game used a 1-bit monochromatic graphical style inspired by games on early Macintosh systems. The chose art style was mind-blowing and showed not all modern games have to look extremely realistic to be amazing.Bellow is a great video which shows off the game “Curse of the Obra Din” with a screenshot from the game.
